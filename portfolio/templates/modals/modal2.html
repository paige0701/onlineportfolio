Kamperkorea-Flask

프로젝트명 KAMPERKOREA-flask
설명 flask를 이용하여 ecommerce website 구현
참여인원 2명
개발환경 및 개발도구

O.S : 1.3 GHz Intel Core i5, MacOS

Framework : Flask

Database : mySQL, sqlalchemy

Tool : Pycharm, MySQL Workbench, SourceTree, AWS

Language : Python, javascript, bootstrap, jquery, css, html

Browser : Chrome
담당업무 50%
개발기간 15주
주요기능

1. 회원가입 및 로그인/로그아웃 기능

2. GoogleAPI를 이용한 위치 파악

3. 장바구니

4. 결제

5. 하우징 예약

6. 관리자 페이지(회원관리, 상품 등록, 하우징 등록, 통계 등)
나의 역할

1. 메인페이지 : 화면구성 기획 및 디자인

2. 디테일 페이지 : 결제모듈 연결, 장바구니 항목 업데이트

3. 카테고리 : 화면구성 및 디자인

4. 지도 : Ajax로 통신, 지도 드레그 시 범위 안의 아이템 로드 (googlemap API이용)

5.페이팔 결제 : Paypalsdk 를 이용, 신용카드 결제기능 구현
