Kamperkorea-django

프로젝트명 KAMPERKOREA-django
설명 오픈소스인 django-oscar를 이용하여 만든 쇼핑몰
참여인원 3명
개발환경 및 개발도구

O.S : 1.3 GHz Intel Core i5, MacOS

Framework : django

Database : mySQL

Tool : Pycharm, MySQL Workbench, SourceTree, AWS

Language : Python, javascript, bootstrap, jquery, css, html

Browser : Chrome
담당업무 40%
개발기간 12주
주요기능

1. 회원가입 및 로그인/로그아웃 기능

2. 페이스북 로그인

3. 장바구니

4. 결제

5. 관리자 페이지(회원관리, 상품 등록, 통계 등)
나의 역할

1. 페이스북 로그인 : Social Auth 를 이용한 페이스북 로그인

2. 디테일 페이지 : 상품 디테일 페이지, 장바구니에 업데이트

3. 장바구니 : 장바구니 업데이트 및 삭제

4. 마이페이지 : 비밀번호 및 사용자 정보 수정

5. 히스토리 페이지 : 결제완료 목록 확인
